# GitHub Traffic boundary

Local Git history can provide commits, authors, additions/deletions, file churn, tags, branches, language mix, and repo health signals.

Local Git history cannot provide:

- repository views;
- unique visitors;
- clone counts from GitHub;
- referrers;
- popular content on GitHub.

The GitHub connector is explicit, optional, token-based, and documented as server-side GitHub analytics collection. It must not be represented as local inference.
